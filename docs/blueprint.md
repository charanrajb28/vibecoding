# **App Name**: CodeSail

## Core Features:

- User Authentication: Secure user authentication and authorization with email/password and OAuth options.
- Dashboard: Personalized dashboard displaying user projects with search, sort, and filter.
- Project Cards: Visually appealing project cards with metadata, status indicators, and quick actions (edit, run, delete).
- Template Selection: Choose from a variety of starter templates (Next.js, React, etc.) when creating a new project.
- Git Import: Import projects directly from Git repositories (GitHub, GitLab, Bitbucket).
- Code Editor: VS Code-like integrated development environment (IDE) with syntax highlighting, autocompletion, and themes.
- Generative AI code tool: Tool in the IDE which offers code completion and improvements, along with explanations. Uses an LLM tool to determine if changes are helpful or detrimental to the project's correctness. Includes options for refactoring suggestions.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to evoke trust, authority, and intelligence, fitting for a development tool.
- Background color: Light gray (#EAEAF2), providing a clean, distraction-free coding environment with sufficient contrast for code.
- Accent color: Soft violet (#8E44AD), used for interactive elements and highlights, maintaining harmony with the primary color.
- Body and headline font: 'Inter', a sans-serif for a modern, neutral, readable interface.
- Code font: 'Source Code Pro' for the IDE.
- Use a set of clean, modern icons from a library like Font Awesome or Material Design, to ensure consistency.
- Clean and organized layout, providing ample screen space for code editing, previews, and debugging.
- Use subtle transitions and animations to enhance UX.
